integration

Change one line. That's it.

Your existing OpenAI SDK code works as-is. Swap the base URL, and PORDL handles the rest — classifying each request and routing it to the most cost-effective model.

BEFORE

  apiKey: 'sk-your-openai-key',
});

// Every request → GPT-4o

AFTER

  apiKey: 'pd_live_your-key',
