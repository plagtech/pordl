Your existing OpenAI SDK code works as-is. Swap the base URL and PORDL handles the rest.

BEFORE

  apiKey: 'sk-your-openai-key',

Smart routing

Simple tasks go to fast, cheap models. Complex tasks go to powerful ones. You save on every call without losing quality.

Creative routing mode

Drafting fiction? Creative mode picks models for prose quality and never escalates to frontier pricing. Long context keeps whole manuscripts in memory.

Auto failover

If a model is down, PORDL routes to the next best option. Your app stays up, no code changes needed.

Savings dashboard

Track exactly how much you're saving in real time. See per-model breakdowns, cache hit rates, and daily cost comparisons.

OpenAI compatible

Same request format, same response format. Works with every OpenAI SDK and library. Python, Node, Go, Rust — all of them.

Flat subscription

Predictable monthly pricing with a hard cap — when credits run out, requests pause. We never auto-charge; top up only if you choose to.

Response caching

Identical requests return cached results instantly at zero credit cost. Cached responses live at most one hour, keyed by one-way hashes.

Built-in content safety

Every request passes an automated content-safety check before reaching any provider. Prohibited content is refused — see the AUP.
